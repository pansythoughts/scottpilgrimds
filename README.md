# Scott Pilgrim DS (EARLY CONCEPT)
a sort of demake/recreation of Scott Pilgrim vs. The World: The Game for the Nintendo DS.

note that this is a very early concept of what could be a 'port' of the original game for this handheld, and therefore, is unfinished. (I just wanted to share it :D.)

## Compiling
unfortunately, as of now, this repo only contains the source of the game and not any assets, since I don't want to add any copyrighted material to this repo.
maybe at some point I could make an asset extraction tool, from the original game files of the PC version, so that to compile this an original copy of the game is required, but I'm still very new to this. **so, any insight on how I could do that is totally welcome.**

if you wish to try this out, check this [GBAtemp thread](https://gbatemp.net/threads/wip-scott-pilgrim-ds-demake.674163/).
